## Usage

Modals are used for focused tasks or important alerts.

```tsx
import AnimatedItemsModal from '@/components/ui-component/modal/AnimatedItemsModal'

export default function Page() {
  return (
    <div className="p-4">
        <AnimatedItemsModal />
    </div>
  )
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| isOpen | boolean | - | Whether the modal is open |
| onClose | () => void | - | Callback when closing |
