title: Badge
description: Small status descriptors for UI elements.
---

## Usage

Badges are used to highlight status, items count, or text classification.

```tsx
import { Badge } from '@/components/ui-component/Foundational/Badge/Badge'

export default function Page() {
  return (
    <div className="flex gap-2">
      <Badge variant="neutral">Neutral</Badge>
      <Badge variant="brand">New</Badge>
      <Badge variant="success">Completed</Badge>
    </div>
  )
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| children | ReactNode | - | The content |
| variant | 'neutral' \| 'brand' \| 'success' | 'neutral' | Visual style |
| className | string | - | Optional classes |
