## Overview

A comprehensive set of navigation headers ranging from simple utility bars to complex mega-menus and e-commerce tailored layouts. All components are built with Tailwind CSS and are fully responsive.

### 1. Simple Header
A clean, minimal header with logo, navigation links, and a primary action button. Includes a sliding mobile menu.

<div className="border border-neutral-200 dark:border-neutral-800 rounded-xl overflow-hidden shadow-sm my-6 bg-white dark:bg-neutral-950">
    <div className="relative h-[300px] bg-neutral-100 dark:bg-neutral-900 overflow-y-auto">
        <HeaderSimple />
         <div className="p-8 text-center text-neutral-400">Scroll or resize to test responsiveness</div>
    </div>
</div>
