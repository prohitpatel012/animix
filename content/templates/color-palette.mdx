# 100+ Hex Color Palette

A comprehensive collection of hand-picked colors ready for use in your web projects. Click any color card to copy its Hex code to your clipboard.

<ColorGrid />

## Usage

These are standard hex codes that can be used in CSS, Tailwind config, or inline styles.

```css
/* CSS Example */
.my-element {
  background-color: #3b82f6; /* Blue */
  color: #ffffff;
}
```

```js
// Tailwind Config Example
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#3b82f6',
        secondary: '#64748b',
      }
    }
  }
}
```
