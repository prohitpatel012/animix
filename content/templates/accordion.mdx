# Accordions

Organize large amounts of content into collapsible sections. Ideal for FAQs, product details, or step-by-step guides.

## Simple
Clean text-only expanders.

<div className="my-8 border rounded-lg overflow-hidden">
  <AccordionSimple />
</div>

```tsx

export default function Example() {
  return <AccordionSimple />;
}
```
