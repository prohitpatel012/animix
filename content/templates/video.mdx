# Video Sections

Video content is one of the most powerful ways to communicate your message. Use these components to embed players, create background atmosphere, or build learning experiences.

## Simple Embed
Standard 16:9 responsive video wrapper.

<div className="my-8 border rounded-lg overflow-hidden">
  <VideoSimple />
</div>

```tsx
import { VideoSimple } from '@/components/ui-component/video';

export default function Example() {
  return <VideoSimple />;
}
```
